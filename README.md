# DicomWebViewer
A Angular SPA web site which used to view DICOM files, use ASP.NET core in the backend, can deploy to Linux platform.

## Used Technologies
- ASP.NET Core/.Net Core 2.2
- Angular 7
- bootstrap
- cornerstone
- log4net (suport .NET Core)
- fo-dicom (support .NET Core)

## Supported Web Host
- Windows + IIS
- CentOS + nginx
